# 青岛凯丰德包装官网

这里保存的是青岛凯丰德包装有限公司官网的最新版源代码，`main` 分支始终与正式网站保持同步。

## 直接下载最新版

### [⬇️ 点击下载最新网页源代码（ZIP）](https://github.com/zxl134679134679-cmd/kfd2zhongwen/archive/refs/heads/main.zip)

这个下载地址始终指向 `main` 分支。以后打开本仓库，点击上面的链接，就会下载当时最新的完整网页代码，无需在文件列表中查找版本。

也可以点击仓库右上方绿色的 **“代码”** 按钮，再选择 **“下载 ZIP”**，下载结果与上面的快捷入口相同。

## 在线网站

[https://en.kfdpack.com/](https://en.kfdpack.com/)

## 项目说明

- Vite + React
- 支持中文与英文切换
- 包含产品、包装方案、工厂实力和资质认证页面
- 询价表单预留 n8n Webhook

## 本地打开

解压下载的文件后，在项目目录安装依赖并启动预览：

```bash
pnpm install
pnpm dev
```
